## 💡 使用提示

### 修改内容
所有页面文本都支持多语言，修改步骤：

1. **找到翻译键**
```html
<h1 data-i18n="page.title">标题</h1>
```

2. **在 lang.js 中修改**
```javascript
zh: { page: { title: '新标题' } }
en: { page: { title: 'New Title' } }
```

### 添加新页面
1. 复制 `_page_template.html`
2. 修改标题和内容
3. 在 `components.js` 导航中添加链接
4. 在 `lang.js` 中添加翻译

### 自定义样式
编辑 `css/style.css`：
```css
/* 在文件底部添加自定义样式 */
.my-custom-style {
    /* 你的样式 */
}
```
